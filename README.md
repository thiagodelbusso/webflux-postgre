# WebFlux - Postgre
### Scripts para criação das tabelas

```sql
CREATE TABLE public.departments
(
    dept_no text COLLATE pg_catalog."default" NOT NULL,
    dept_name text COLLATE pg_catalog."default",
    CONSTRAINT departments_pkey PRIMARY KEY (dept_no)
)
```

```sql
CREATE TABLE public.employee
(
    emp_id integer NOT NULL GENERATED BY DEFAULT AS IDENTITY ( INCREMENT 1 START 1 MINVALUE 1 MAXVALUE 2147483647 CACHE 1 ),
    nome text COLLATE pg_catalog."default",
    idade integer,
    id_department text COLLATE pg_catalog."default",
    CONSTRAINT employee_pkey PRIMARY KEY (emp_id),
    CONSTRAINT fk_customer FOREIGN KEY (id_department)
        REFERENCES public.departments (dept_no) MATCH SIMPLE
        ON UPDATE NO ACTION
        ON DELETE NO ACTION
)
```
